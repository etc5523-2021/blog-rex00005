This article will statistically explore the coronavirus situations in the middle east area and especially in the United Arab Emirates, as well as how the United Arab Emirates fought this pandemic.

Data description

The data used for this article is obtained from Our World in Data.

The data set contains 117021 rows and 62 columns of data. After the data cleaning, the variables used is shown in the following data dictionary.

Table 1: Data dictionary
Variable Class Description
date date Date of observation
location double Countries
total_cases double Total confirmed cases
new_cases double Newly confirmed cases
total_deaths double Total recorded deaths
new_deaths double Newly recorded deaths
total_tests_per_thousand double Total number of tests per thousand
total_vaccinations double Total number of vaccination
people_vaccinated double Number of people has one vaccine
people_fully_vaccinated double Number of people fully vaccined
